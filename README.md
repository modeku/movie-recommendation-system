# 基于多模态数据的个性化电影推荐系统
🎓 本科毕业设计项目

## 项目简介
本项目是一款融合多模态特征的智能电影推荐系统，结合电影海报视觉画面、剧情文本描述、标签类型三大维度数据，解决传统推荐系统的冷启动问题，为用户提供精准的个性化电影推荐服务。

## 技术栈
| 模块 | 技术选型 |
|------|----------|
| 前端 | Vue3、Axios、Element Plus |
| 后端 | Python、FastAPI |
| 模型 | PyTorch、ResNet50（图像特征）、BERT（文本特征） |
| 数据库 | MySQL |
| 数据集 | MovieLens 1M、TMDB、Kaggle 公开电影数据集 |

## 核心功能
1. 免密码极简注册登录
2. 三步快速构建用户冷启动画像（热门推荐、海报视觉偏好、电影类型选择）
3. 多模态特征融合相似度计算
4. 个性化首页智能电影推荐
5. 电影详情浏览、偏好收藏

## 项目亮点
✅ 突破传统单一推荐逻辑，多模态融合提升推荐精准度
✅ 完美解决新用户冷启动痛点
✅ 前后端分离架构，代码模块化，可二次开发
✅ 完整的从数据处理、模型训练、接口联调到页面展示的全工程落地流程
